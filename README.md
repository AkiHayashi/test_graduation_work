# 開発言語

+ Ruby2.6.5

+ Rails5.2.5

# 就業Termの技術

+ devise

+ お気に入り機能

# カリキュラム外の技術

+ chartkick

+ Cocoon

# 実行手順

```
$ git clone git@github.com:AkiHayashi/graduation_work.git
$ cd kenkoutecho
$ bundle install
$ rails db:create && rails db:migrate
& rails s
```
